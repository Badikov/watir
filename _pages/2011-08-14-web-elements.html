All HTML elements are supported by Watir-WebDriver. Some examples are below:

Text Fields

[sourcecode language="ruby" light="true"]
require 'watir-webdriver'
b = Watir::Browser.start 'bit.ly/watir-webdriver-demo'
t = b.text_field :id => 'entry_0'
t.exists?
t.set 'your name'
t.value
[/sourcecode]

Select Lists - Combos

[sourcecode language="ruby" light="true"]
require 'watir-webdriver'
b = Watir::Browser.start 'bit.ly/watir-webdriver-demo'
s = b.select_list :id => 'entry_1'
s.select 'Ruby'
s.selected_options
[/sourcecode]

Radios

[sourcecode language="ruby" light="true"]
require 'watir-webdriver'
b = Watir::Browser.start 'bit.ly/watir-webdriver-demo'
r = b.label(:text => 'What is ruby?').parent.radio :value => 'A gem'
r.exists?
r.set
r.set?
[/sourcecode]

Checkboxes

[sourcecode language="ruby" light="true"]
require 'watir-webdriver'
b = Watir::Browser.start 'bit.ly/watir-webdriver-demo'
c = b.label(:text => 'What versions of ruby?').parent.checkbox :value => '1.9.2'
c.exists?
c.set
c.set?
[/sourcecode]

Buttons

[sourcecode language="ruby" light="true"]
require 'watir-webdriver'
b = Watir::Browser.start 'bit.ly/watir-webdriver-demo'
btn = b.button :value => 'Submit'
btn.exists?
btn.click
[/sourcecode]

Links

[sourcecode language="ruby" light="true"]
require 'watir-webdriver'
b = Watir::Browser.start 'bit.ly/watir-webdriver-demo'
l = b.link :text => 'Google Docs'
l.exists?
l.click
[/sourcecode]

Divs & Spans

[sourcecode language="ruby" light="true"]require 'watir-webdriver'
b = Watir::Browser.start 'bit.ly/watir-webdriver-demo'
d = b.div :class => 'ss-form-desc ss-no-ignore-whitespace'
d.exists?
d.text
s = b.span :class => 'ss-powered-by'
s.exists?
s.text
[/sourcecode]

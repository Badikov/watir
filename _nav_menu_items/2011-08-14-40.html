Welcome to the home of Watir WebDriver: the most elegant way to use WebDriver with ruby. Watir (pronounced Water: I know it’s confusing) stands for Web Application Testing in Ruby, and the WebDriver part stands for, um, WebDriver. What browsers does Watir-WebDriver support? Any browser that WebDriver supports: namely Firefox, Chrome & IE. Safari support is sadly missing. How good is Watir-WebDriver? I could make up lots of quotes about how good it is, but I don’t need to, it’s that good. Getting Started gem install watir-webdriver or just add watir-webdriver to your gemfile if you rock and use bundler (if you don’t use bundler, you should, right now). Watir-WebDriver ♥s ruby 1.9.2, but it won’t complain in older versions nonetheless Getting Going Ready, steady… at a prompt, type irb. Meet Ruby, your servant who will perform any (syntactically correct) command you can throw her way. So type in the following commands one by one (but don’t shout them) to see some amazing things happen in browserland: require ‘watir-webdriver’ b = Watir::Browser.new b.goto ‘bit.ly/watir-webdriver-demo’ b.text_field(:id => ‘entry_0′).set ‘your name’ b.select_list(:id => ‘entry_1′).select ‘Ruby’ b.select_list(:id => ‘entry_1′).selected? ‘Ruby’ b.button(:value => ‘Submit’).click b.text.include? ‘Thank you’ Wow, wasn’t that easy!

Watir exists because a few people got together to share what they were doing. The community has grown and Watir keeps getting better because people who use it continue to share with others. Please join our community. Source Code The Watir Source code is available on github. Please see Building Watir if you want to contribute. Mailing Lists Watir General is a Google group for anyone using Watir for testing. Please read these guidelines before posting to the list. Watir Development is a mailing list for all contributors to the Watir family of tools. Any one interested in Watir development is welcome to join.

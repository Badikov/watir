Text Fields Select Lists – Combos Checkboxes & Radios Buttons & Links Divs & Spans WYSIWYG Editors There are two ways to enter text into a WYSIWYG editor using Watir-WebDriver: locate the iFrame, and use .send_keys to enter text execute javascript on the browser object that sets the value of the WYSIWYG editor CKEditor [sourcecode language="ruby" light="true"] require ‘watir-webdriver’ b = Watir::Browser.new :firefox b.goto ‘http://ckeditor.com/demo’ b.execute_script(“CKEDITOR.instances['editor1'].setData(‘hello world’);”) b.frame(:title => ‘Rich text editor, editor1, press ALT 0 for help.’).send_keys ‘hello world again’ [/sourcecode] TinyMCE Editor [sourcecode language="ruby" light="true"] require ‘watir-webdriver’ b = Watir::Browser.new b.goto ‘http://tinymce.moxiecode.com/tryit/full.php’ b.execute_script(“tinyMCE.get(‘content’).execCommand(‘mceSetContent’,false, ‘hello world’ );”) b.frame(:id => “content_ifr”).send_keys ‘hello world again’ [/sourcecode]
